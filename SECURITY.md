# Security Policy

Castellyn stores provider API keys and a dashboard token in the Windows
Credential Manager and spawns local PowerShell maintenance scripts. Security
issues are taken seriously.

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue.

- Email: **bideppvx@gmail.com**

Include steps to reproduce, the affected version, and any relevant logs. Expect
an initial response within a few days.

## Supported versions

Only the latest released version receives security fixes.
