## Top-10 (простым языком)

1. **R2 — приложение не запоминает, где ты был.** Каждый запуск — снова «Обновления», хотя фича «вернуться на последнюю вкладку» написана и когда-то работала. Технически: код, который *сохраняет* вкладку, срабатывает раньше кода, который её *восстанавливает*, и затирает сохранённое. *Аналогия:* закладка в книге, которую каждый раз кто-то перекладывает на первую страницу. **Фикс:** читать сохранённую вкладку до/внутри инициализации состояния (или guard первого прогона эффекта). *Что сломается:* ничего; риск нулевой.
2. **R3 — «Остановить стек» из трея молча не работает.** Окно свёрнуто → пункт меню шлёт событие → фронт открывает подтверждение в невидимом окне → тишина. *Аналогия:* кнопка звонка, звонящая в запертой комнате. **Фикс:** `reveal(app)` для stack_stop (или пропуск confirm для tray-источника + уведомление). *Blast radius:* один арм в match.
3. **R4 — онбординг теряет введённый путь.** Новичок вводит папку, жмёт «Далее» — путь выброшен, все вкладки пустые. Единственная кнопка, которая сохраняет — «Сохранить», но «Далее» активна и без неё. *Фикс:* `next()` на шаге 1 сам вызывает `saveScripts()` (или `canNext = scriptsSaved`). Это первые 30 секунд опыта нового OSS-пользователя.
4. **R1 — зелёные тосты, которым нельзя верить.** Компонент со статусом `held` или упавший скрипт с exit-0 даёт «Актуально/успешно»; операционные прогоны вообще не читают envelope. *Аналогия:* курьер отчитался «доставлено», не открыв багажник. **Фикс:** ветка `held` в `deriveOutcome` + читать envelope и для операционных id (где он есть).
5. **U1 — горячие клавиши не совпадают с тем, что видишь.** Ctrl+2 ведёт на «Обновления», хотя вторым в сайдбаре стоит «Сессии»; палитра подсказывает те же «неправильные» номера. Порядок хоткеев — отдельный захардкоженный список, сайдбар — другой (+ пользователь может перетасовать его drag-ом). **Фикс:** производить порядок хоткеев из фактического порядка сайдбара (shared store). *Что сломается:* мышечная память тех, кто выучил текущие номера (их, судя по багу, нет).
6. **U2 — «Историю уведомлений» нельзя закрыть по-человечески.** Escape не работает, клик по сайдбару переключает вкладку ПОД панелью, а панель остаётся; висит даже поверх других модалок. **Фикс:** onkeydown Escape + закрытие на любую навигацию/потерю контекста.
7. **V1/V2 — таблицы наезжают и обрезаются.** В MCP подсказка и чипы профилей заезжают под кнопки; в Форках и Плагинах на обычных 1440px появляется горизонтальный скролл и обрезанные кнопки («Клониров…»). **Фикс:** пересмотр min-width колонок/wrap для этих трёх таблиц.
8. **R5/U3 — молчаливые отказы.** Скрипт не запустился — единственный след в свёрнутом логе; кнопки на Главной при занятом слоте просто «не нажимаются» без объяснения. **Фикс:** тост/авто-раскрытие консоли при spawn-ошибке; `disabled + title` на quick actions.
9. **V5 — «1 файлов, 4 плагинов».** Русская плюрализация в новых счётчиках не подключена, хотя plural-механизм в проекте есть и старые места им пользуются. Мелочь, которая сразу бросается в глаза на Главной.
10. **V3/V4 — свежие визуальные регрессии.** Поля новых диалогов не растянуты (Base URL обрезает собственный плейсхолдер); статус апдейтера в светлой теме почти невидим — та же болезнь «сырых цветов», которую только что чинили по всему приложению (V2 из 06-30), воспроизведена в новом коде.
